# OpenSeaStack Website

The official landing page for [OpenSeaStack](https://openseastack.org).

## Tech Stack
- **Vite**: Build tool and dev server.
- **Vanilla CSS**: No framework, just clean, modern CSS variables and glassmorphism.
- **GitHub Actions**: Automated deployment to GitHub Pages.

## Development
1. `npm install`
2. `npm run dev`

## Deployment
Pushing to `main` triggers the `deploy.yml` workflow, which builds the site and deploys to GitHub Pages.
